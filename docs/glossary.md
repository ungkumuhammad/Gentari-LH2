# Glossary

Terminology for the LH2 value chain. Add terms as the repo grows; keep
definitions sourced where a number is involved (`CLAUDE.md` §4).

| Term | Definition |
|------|------------|
| **LH2** | Liquid hydrogen; H2 cooled to its normal boiling point (~20.3 K) and stored as a cryogenic liquid. |
| **GH2 / CGH2** | Gaseous hydrogen / compressed gaseous hydrogen. |
| **SEC** | Specific Energy Consumption — energy per unit product, e.g. liquefaction SEC in kWh/kg. |
| **BOR** | Boil-Off Rate — fraction of stored LH2 that evaporates per day (%/day), at rest or in transit. |
| **BOG** | Boil-Off Gas — vapor generated by heat ingress into LH2. |
| **Ortho/para hydrogen** | Nuclear spin isomers of H2. Ortho→para conversion is exothermic and must be managed during liquefaction to limit later self-boil-off. |
| **NBP** | Normal Boiling Point (at 1 atm); ~20.3 K for H2. |
| **LCOH** | Levelized Cost of Hydrogen — discounted lifetime cost per kg H2 delivered. |
| **CapEx / OpEx** | Capital / operating expenditure. |
| **NPV / IRR** | Net Present Value / Internal Rate of Return. |
| **Electrolysis (PEM/AEL/SOEC)** | Water splitting via Proton Exchange Membrane, Alkaline, or Solid Oxide Electrolysis. |
| **SMR + CCS** | Steam Methane Reforming with Carbon Capture and Storage ("blue" H2). |
| **Reliquefaction** | Re-cooling boil-off gas back to liquid, e.g. onboard a carrier. |
| **Vaporizer / regas** | Equipment converting LH2 back to GH2 at the import terminal. |
| **Well-to-gate / gate-to-gate / well-to-wheel** | System-boundary descriptors for where an analysis starts and ends. |
| **Cold energy recovery** | Useful recovery of the refrigeration released when LH2 is vaporized. |
